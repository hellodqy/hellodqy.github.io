【问题描述】

约瑟夫问题：n个骑士编号1，2，…，n，围坐在圆桌旁。编号为1的骑士从1开始报数，报到m的骑士出列，然后下一个位置再从1开始报数，找出最后留在圆桌旁的骑士编号。

【输入形式】

程序参考的输入（数字前为提示文字）：

Input n and m:7 3

【输出形式】

Result:4

【样例输入】

Input n and m:7 3

【样例输出】

Result:4
